|Definition |Name |Default Value |Description

|PERSISTENCE_CACHE_RETRIEVEMODE
|jakarta.persistence.cache.retrieveMode
|String
|The retrieveMode hint supports the values *USE* and *BYPASS* and tells {product-name} if it shall *USE* the second-level cache to retrieve an entity or if it shall BYPASS it and get it directly from the database.

|PERSISTENCE_CACHE_STOREMODE
|jakarta.persistence.cache.storeMode
|String
|Valid values are USE, BYPASS or REFRESH. If the setting is not recognised, it defaults to USE. The storeMode hint controls the storage and update of the second-level cache.
